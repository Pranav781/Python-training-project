A simple, console-based Python application that integrates three essential campus management modules. This project is built using basic Python concepts, making it a great reference for beginners understanding object-oriented programming, file handling, or structured data manipulation.

!! Features
The application consists of three main modules:

1. **Library Management System**
   * View available books.
   * Issue books to students.
   * Return books to the library.

2. **Student Feedback System**
   * Collect structured feedback for faculty and campus facilities.
   * Store and display submitted feedback reviews.

3. **Course Management System**
   * List available courses and elective tracks.
   * Enroll students into specific courses.
   * View student enrollment details.

---
!! Concepts Covered

This project utilizes fundamental Python programming blocks:
* **Object-Oriented Programming (OOP):** Classes and objects representing Books, Courses, and Students.
* **Data Structures:** Lists and Dictionaries to manage and track real-time records.
* **Control Flow:** Conditional loops (`if-elif-else`) and repetitive execution (`while` loops) for user menus.
* **Functions:** Clean, modular functions keeping code maintainable and reusable.
